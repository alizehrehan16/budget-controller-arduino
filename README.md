Overview
This project implements a simple embedded budgeting system using an Arduino UNO and basic digital components.

The system tracks a running balance using two push buttons:
• Income button increases the balance
• Expense button decreases the balance

Three LEDs indicate the current financial state using threshold logic:
• Green – balance is healthy
• Yellow – caution zone
• Red – spending limit exceeded

Features:

* Real-time balance updates
* LED threshold state machine
* Serial Monitor logging
* Two-button input interface
* Breadboard prototype hardware design

Skills Demonstrated:

* Arduino programming (C/C++)
* Digital input/output control
* Hardware debugging
* Breadboard prototyping
* Embedded system logic design
